PG핵 오류해결 방법

error code = hk0820-2132

해결 방법 : PGhelper라는 폴더에 들어가 PGhelper이라는 프로그램을 실행시키고 3초정도 기다리시면 오류가 고쳐 집니다.
#방화벽을 모두 해제한 상태에서만 고쳐 집니다. 아니면 바이러스로 인식하고 PGhelper를 지웁니다.#
간혹가다 오류가 안고쳐지는 버그가 있습니다. 그런경우에는 유료버전을 구매 하셔야 합니다. 사이트에 들어가시면
이메일 주소가 있는데 문화상품권 5000원 핀번호를 보내주시고 자신이 받을 이메일 주소를 첨부 해주시면 저희가
확인후 유료버전 Beta버전이 아닌 Full버전을 보내드리겠습니다.

error code = hp9029-2123

해결 방법 : 윈도우가 10 이하 버전인지 확인을 합니다. 윈도우 10 이상버전에서만 작동합니다. 윈도우 8, 9, 10버전에서 작동하는 핵은 Full버전인데
Full버전은 유료입니다. 사이트에 들어가시면
이메일 주소가 있는데 문화상품권 5000원 핀번호를 보내주시고 자신이 받을 이메일 주소를 첨부 해주시면 저희가
확인후 유료버전 Beta버전이 아닌 Full버전을 보내드리겠습니다.

#위 에러 코드말고는 확인된 에러 사항이 없습니다. 사이트에 있는 이메일로 에러를 보내주시면 무료로 Full버전을 전달해 드리겠습니다. #
